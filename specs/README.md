# XSC 功能规范文档

本文档使用 Gherkin 语法（Cucumber/Behavior Driven Development）描述 XSC 项目的功能规范。

## 文件说明

- **xsc.feature** - 完整的功能规范，包含所有用户场景

## 规范结构

规范分为以下几个功能模块：

### 1. TUI 模式下的 SSH 会话管理
- **基础导航** - Vim 风格导航、命令模式跳转
- **目录操作** - 展开/折叠、全部展开/折叠
- **搜索功能** - 实时搜索、搜索退出、清除搜索
- **会话操作** - 连接、新建、编辑、删除
- **信息显示** - 详情面板、帮助、状态栏
- **退出程序** - :q、Ctrl+c 等退出方式
- **错误处理** - 错误弹窗、连接失败处理

### 2. 命令行模式
- 列出所有会话
- 连接到指定会话
- 模糊匹配连接
- 导入 SecureCRT 会话

### 3. 会话配置管理
- 本地会话加载
- 三种认证方式（密码、密钥、Agent）
- SecureCRT 延迟解密

### 4. 全局配置管理
- SecureCRT 集成配置
- SSH 主机密钥验证配置

### 5. TUI 界面元素
- 行号显示
- 详情边框
- 状态栏信息
- 搜索状态提示
- 实时密码解密

### 6. 性能优化
- SSH Agent 密钥缓存
- 延迟解密
- 高效树形渲染

## 使用方式

这些规范可以用于：

1. **开发指导** - 了解需要实现的功能
2. **测试用例** - 基于规范编写自动化测试
3. **文档** - 作为功能说明文档
4. **验收标准** - 定义功能完成的标准

## 关键术语

- **TUI** - Terminal User Interface，终端用户界面
- **Session** - SSH 会话配置
- **Node** - 树形结构中的节点（目录或会话）
- **Fold** - 目录的展开/折叠状态
- **Filter** - 搜索过滤条件
- **Agent** - SSH Agent，用于密钥管理

## 快捷键速查

### 导航
- `j/k` - 下/上移动
- `gg/G` - 第一行/最后一行
- `nG/:n` - 跳转到第 n 行

### 目录
- `o` - 切换展开/折叠
- `h/l` - 折叠/展开
- `E/C` - 展开/折叠所有

### 搜索
- `/` - 进入搜索
- `Enter` - 确认搜索
- `Esc` - 清空搜索

### 操作
- `Enter` - 连接会话
- `n` - 新建会话
- `e` - 编辑会话
- `c` - 重命名会话
- `D` - 删除会话（需输入 YES 确认）
- `?` - 显示帮助

### 退出
- `:q` - Vim 风格退出
- `Ctrl+c` - 强制退出
